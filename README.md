# UE5-Game
My first UE5 project.

I created a game with one gameplay stage that includes elements such as:
- combat (melee weapon, bow, magic),
- AI ncp
- statistics,
- inventory,
- quests,
- dialogue,
- trade,
- save/load,
- customizable key bindings.
- All systems are connected and working.
